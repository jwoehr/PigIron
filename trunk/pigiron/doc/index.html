PigIron Open Source Java Classes for VSMAPI

Project Goal

The goal of the project is to provide Java class wrappers for the IBM z/VM Virtual Machine Operating System Systems Management Application Programming Interface (sometimes called VSMAPI).

Additionally, there are visual widgets planned based on class embodiments of VSMAPI functions so that VSMAPI functions can be dragged and dropped into a yet-to-be-defined workspace and executed individually or in ordered collections as if they were scripts.

Please visit PigIron on Source Forge to download source, executable code and documentation.
