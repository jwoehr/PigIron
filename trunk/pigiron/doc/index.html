Overview

The goal of the project is to provide Java class wrappers for the IBM z/VM Virtual Machine Operating System Systems Management Application Programming Interface (sometimes called VSMAPI).

Note: PigIron is not associated with the IBM corporation.

NOTE ON SECURITY (LACK THEREOF)

Currently there is NO secure sockets implementation. Sends your password in CLEAR TEXT. Use ONLY on a secure LAN behind a firewall.

Current Level of Support

PigIron currently supports the z/VM R5.3 level of VSMAPI.

Future Development Plans

Additionally to the basic class library, there are visual widgets planned based on PigIron's class embodiments of VSMAPI functions so that VSMAPI functions can be dragged and dropped into some yet-to-be-defined workspace and executed individually or in ordered collections as if they were scripts.

Getting PigIron

Please visit PigIron on Source Forge to download source, executable code and documentation.
