PigGen Paramstruct Java AutoGenerator

Overview of PigIron's PigGen

PigGen autogenerates Java source code to embody PigIron-specific VSMAPI parameter structures for either native VSMAPI calls or for your own custom coding (assuming that someone is custom coding the VSMAPI host side that will answer your custom function calls).
