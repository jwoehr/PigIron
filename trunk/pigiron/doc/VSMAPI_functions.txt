List of VSMAPI Functions Noting Test Coverage
-----------------

The classes listed below represent all PigIron's implemented VSMAPI Functions.

Many have been tested through their ObjectRexx interface found in

    pigiron/script/oorexx/functest/<FunctionName>.cls

invoked via its corresponding test driver

    pigiron/script/oorexx/functest/<FunctionName>.rex

from the script

    pigiron/script/oorexx/functest/allfunctest.rex

or through the FIJI interface generated by PigGen and found in

    pigiron/script/fiji/pigiron_support.fiji

along with the test driver

    pigiron/script/fiji/allfunctest.fiji

These test functions are marked with a *. Examine allfunctest.rex and
allfunctest.fiji etc. to understand what is covered by the tests and
what is not. The tests are indicative but not exhaustive.
