PigIron Java Class Library for VSMAPI Programming

Purpose

PigIron is an open source Java library implementing the client to IBM ® z/VM ® V6R3.0 Systems Management Application Programming transactions over TCP/IP sockets (referred to in PigIron documentation as VSMAPI).
