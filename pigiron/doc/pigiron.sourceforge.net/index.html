PigIron Open Source Java Class Libraries for VSMAPI

PigIron is an open source Java class library which:

  • provides Java class wrappers for the IBM ® z/VM ® Virtual Machine Operating System's Systems Management Application Programming Interface
    • (called VSMAPI in PigIron documentation)
  • is extensible by the end user without deep knowledge of Java
Additional Features:
  • PigGen autogenerates VSMAPI parameter type and function from m4-based description files
  • Complete scripting interface using BSF4Rexx on top of Open ObjectRexx to run the PigIron Java classes.
  • Incomplete scripting interface using FIJI ForthIsh Java Interpreter
  • PigLet the PigIron Servlet when completed will provide VSMAPI functionality as JSON data stream for Web applications.

Current development state: Alpha

  • Code in tree supports all 99 VSMAPI for z/VM V5R3 functions and the 15 additional VSMAPI for z/VM V5R4 functions.
    • Not Fully Tested
    • To the extent that we have tested, we have tested on
      • OpenBSD
      • OpenSolaris x86
      • Linux s390x
